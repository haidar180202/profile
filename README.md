## 🚀 AVAILABLE COMMANDS & SCRIPTS (`HOW TO RUN LOCALLY`)

In the project directory (`profile`), you can run the following commands:

### **1. Launch Development Web Server (`npm start`)**
Runs the interactive React portfolio application in development mode.
Open [http://localhost:3000](http://localhost:3000) in your browser to view and interact with the live UI. The page will hot-reload upon making modifications within `src/components/` or `src/data/`.

### **2. Generate Executive PowerPoint Presentation (`node generate-portfolio-ppt.js`)**
Executes the Node.js automation pipeline to compile and generate **`Project_Development_Portfolio_Haidar_v5_Master.pptx`** from structured project datasets, ensuring presentation slides remain perfectly synchronized with source code achievements.

### **3. Build Production Web Bundle (`npm run build`)**
Builds the optimized production application to the `build/` directory. Correctly bundles React 18 in production mode, minifies asset bundles, and attaches unique content hashes for maximum loading speeds and CDN deployment readiness.

### **4. Run Unit & Component Tests (`npm test`)**
Launches the Jest and React Testing Library (`RTL`) test runner in interactive watch mode to verify component rendering fidelity and data collection integrity.
